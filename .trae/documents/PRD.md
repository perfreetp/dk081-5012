## 1. 产品概述

「搬好家」是一款面向毕业季学生、宿舍搬离人群和校园二手群管理员的极简网页应用，解决校内二手家具转手时拆装配送的协调难题。把原本需要反复沟通确认的搬运规则、拼车配送、拆装难度、交接确认等信息，压缩到一次发布里，实现轻量、快用、低沟通成本的家具流转体验。

- 目标用户：即将离校的毕业生、需要更换宿舍的学生、校园二手交易群管理员
- 核心价值：一次发单搞定家具转手全流程，消除信息差和重复沟通

## 2. 核心功能

### 2.1 用户角色

| 角色 | 进入方式 | 核心权限 |
|------|----------|----------|
| 普通学生 | 手机号/学号登录 | 发单、拼单、浏览、议价、评价 |
| 群管理员 | 手机号/学号登录 + 管理员标记 | 代发布多单、管理拼单、查看片区统计 |

### 2.2 功能模块

1. **发单页**：快速发起家具转手需求，选择场景/物品/配送方式，自动提示校区规则
2. **拼单页**：按日期和片区聚合订单，同车拼配送费，降低单人搬运成本
3. **校区规则页**：各宿舍楼限时搬运、货车通行、门岗登记等规则速查
4. **进度页**：订单状态追踪、交接码验证、离校待办清单、完工评价

## 3. 核心流程

### 3.1 发单流程
用户进入发单页 → 选择转手场景 → 填写物品信息（类型、拆装难度、照片）→ 选择位置与离校日期 → 系统自动提示该楼搬运规则 → 选择配送方式 → 设置标签与价格 → 提交发单 → 系统生成订单并匹配拼单池

### 3.2 拼单流程
发单成功后订单进入拼单池 → 同日同片区订单自动聚合 → 用户可在拼单页查看并加入 → 达到拼车人数后确认发车 → 配送费自动分摊

### 3.3 交接流程
配送开始 → 买卖双方获得交接码 → 师傅到达 → 买家出示交接码/卖家确认 → 完成交接 → 生成待办清单 → 评价反馈

## 4. 用户界面设计

### 4.1 设计风格
- **主色调**：暖橙色 #FF6B35（活力、搬家、温暖的暗示）+ 深灰 #1A1A2E（稳重、信息密度）
- **辅助色**：薄荷绿 #2EC4B6（状态提示）+ 柔黄 #FFD166（标签高亮）+ 淡粉 #FF8FA3（公益标签）
- **按钮风格**：圆角矩形（12px），主按钮实心暖橙，次按钮描边，按压有微缩放动画
- **字体**：标题用 Noto Sans SC Bold，正文用 Noto Sans SC Regular，数字用 DM Mono
- **布局风格**：卡片式布局，顶部标签导航，单列信息流，卡片间留白充足
- **图标风格**：线性图标（Stroke 1.5px），搭配 emoji 辅助说明家具类型

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI元素 |
|----------|----------|--------|
| 发单页 | 场景选择 | 三张大卡片横排，选中态橙色边框+放大微动效，图标+文字 |
| 发单页 | 物品信息 | 可多选的家具类型标签云，拆装难度三级选择器（易/中/难），照片上传区 |
| 发单页 | 位置与时间 | 下拉选择校区/宿舍楼，日期选择器，规则提示卡片自动弹出 |
| 发单页 | 配送方式 | 两选一卡片（自提/配送），地址输入框，地图预览缩略 |
| 发单页 | 标签定价 | 勾选标签胶囊，价格输入框，议价开关，发布按钮固定底部 |
| 拼单页 | 拼单列表 | 日期分组的时间线布局，每条拼单卡片含片区、人数、费用分摊 |
| 拼单页 | 拼单详情 | 底部抽屉弹出，展示订单列表和物品汇总 |
| 校区规则页 | 规则卡片 | 按楼分组的折叠卡片，高亮限时时段，图标标注规则类型 |
| 进度页 | 订单状态 | 竖向时间线，节点图标变色表示进度，交接码居中大字展示 |
| 进度页 | 待办清单 | 勾选式清单，离校日倒计时顶部显示 |
| 进度页 | 评价 | 星级评分+标签快捷评价，提交后淡出 |

### 4.3 响应式设计
- 桌面优先设计，最大宽度 480px 居中（移动端体验优先，因为用户多为手机使用）
- 桌面端展示为居中手机框样式，左右留白
- 触摸优化：按钮最小 44px 点击区域，卡片滑动操作

### 4.4 动效设计
- 页面切换：左右滑动过渡
- 卡片交互：按压缩放 0.97，松开回弹
- 交接码生成：数字滚动动画
- 拼单成功：粒子撒花效果
- 状态更新：节点脉冲光圈动画
